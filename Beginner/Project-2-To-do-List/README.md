# 📋 مدیریت لیست کارها (To-Do List)

یک برنامه‌ی ساده برای مدیریت کارهای روزمره با قابلیت ذخیره‌سازی در فایل.

## ✨ ویژگی‌ها
- افزودن کار جدید
- حذف کار
- ویرایش عنوان کار
- علامت زدن کار به عنوان انجام‌شده
- نمایش لیست کارها
- نمایش آمار (تعداد کل، انجام‌شده، در انتظار)
- ذخیره‌سازی خودکار در فایل JSON
- مدیریت خطا
