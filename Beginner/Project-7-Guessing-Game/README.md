# 🎯 بازی حدس عدد (Number Guessing Game)

یک بازی ساده که در آن کامپیوتر یک عدد تصادفی انتخاب می‌کند و شما باید آن را حدس بزنید.

## ✨ ویژگی‌ها
- تولید عدد تصادفی بین ۱ تا ۱۰۰
- راهنمایی (بزرگ‌تر یا کوچک‌تر)
- محدودیت ۱۰ بار حدس
- ذخیره‌سازی بهترین رکورد
- امکان بازی مجدد
