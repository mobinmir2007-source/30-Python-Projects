# 🔐 تولید‌کننده‌ی رمز عبور

یک برنامه‌ی ساده و قدرتمند برای تولید رمزهای عبور قوی و امن.

## ✨ ویژگی‌ها
- تولید رمز عبور تصادفی با طول دلخواه
- تنظیمات سفارشی شامل:
  - حروف بزرگ (A-Z)
  - حروف کوچک (a-z)
  - اعداد (0-9)
  - نمادها (!@#$%...)
- نمایش قدرت رمز (ضعیف، متوسط، قوی)
- رابط کاربری ساده و روان
