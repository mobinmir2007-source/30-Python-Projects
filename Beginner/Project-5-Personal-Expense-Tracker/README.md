# 💰 مدیریت هزینه‌های شخصی

یک برنامه‌ی کامل برای مدیریت درآمد و هزینه‌های شخصی.

## ✨ ویژگی‌ها
- ثبت درآمد و هزینه
- نمایش همه‌ی تراکنش‌ها
- نمایش خلاصه (جمع درآمد، جمع هزینه، مانده)
- گزارش ماهانه
- جستجو در تراکنش‌ها
- حذف تراکنش
- ذخیره‌سازی خودکار در فایل JSON
