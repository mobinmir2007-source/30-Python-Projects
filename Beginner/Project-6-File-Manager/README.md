# 📁 مدیریت فایل‌ها (File Manager)

یک برنامه‌ی ساده برای مدیریت فایل‌های متنی از طریق خط فرمان.

## ✨ ویژگی‌ها
- ساخت فایل جدید با پسوند `.txt`
- نوشتن در فایل
- خواندن فایل
- حذف فایل
- مشاهده‌ی لیست فایل‌ها
- نمایش حجم فایل‌ها
- ذخیره‌سازی لیست فایل‌ها در JSON

## 🚀 نحوه‌ی اجرا
```bash
python main.py
