# 🌤️ برنامه‌ی نمایش وضعیت آب و هوا

یک برنامه‌ی ساده برای دریافت و نمایش وضعیت آب و هوا با استفاده از OpenWeatherMap API.

## ✨ ویژگی‌ها
- دریافت آب و هوای هر شهر
- نمایش دما (سانتی‌گراد)
- نمایش رطوبت
- نمایش وضعیت آب و هوا
- نمایش سرعت باد
- ذخیره‌سازی تاریخچه
- مدیریت تاریخچه

## 🔑 دریافت کلید API
1. به [OpenWeatherMap](https://openweathermap.org/api) بروید
2. ثبت‌نام کنید و کلید بگیرید
3. در `main.py` جایگزین کنید: `self.api_key = "YOUR_API_KEY"`

## 🚀 نحوه‌ی اجرا
```bash
pip install requests
python main.py
