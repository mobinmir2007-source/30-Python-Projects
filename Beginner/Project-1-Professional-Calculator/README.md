# 🧮 ماشین‌حساب حرفه‌ای

یک ماشین‌حساب پیشرفته در ترمینال با قابلیت‌های ویژه.

## ✨ ویژگی‌ها
- ۷ عمل ریاضی (جمع، تفریق، ضرب، تقسیم، توان، ریشه، درصد)
- حافظه (M+) برای ذخیره‌سازی اعداد
- تاریخچه‌ی کامل محاسبات
- مدیریت خطا (تقسیم بر صفر، ورودی نامعتبر)
- رابط کاربری زیبا و ساده
